详细信息

QJSValueIterator 类为QJSValue提供了一个java风格的迭代器。

QJSValueIterator 构造函数接受一个 QJSValue 作为参数。构造完成后，迭代器位于属性序列的最开始。下面是如何迭代QJSValue的所有属性:


  QJSValue object;
  ...
  QJSValueIterator it(object);
  while (it.hasNext()) {
      it.next();
      qDebug() << it.name() << ": " << it.value().toString();
  }

next()将推进迭代器。name() 和 value() 函数返回最后跳过的项的名称和值。

注意，QJSValueIterator 只迭 QJSValue自己的属性;也就是说，它不遵循原型链。你可以使用这样的循环跟随原型链:


  QJSValue obj = ...; // the object to iterate over
  while (obj.isObject()) {
      QJSValueIterator it(obj);
      while (it.hasNext()) {
          it.next();
          qDebug() << it.name();
      }
      obj = obj.prototype();
  }

参见QJSValue::property()。

成员函数文档

QJSValueIterator::QJSValueIterator(const QJSValue &object)

构造一个遍历object（对象）的迭代器。迭代器被设置在属性序列的前面(在第一个属性之前)。

QJSValueIterator::~QJSValueIterator()

销毁迭代器。

bool QJSValueIterator::hasNext() const

如果迭代器前面至少有一个项(即迭代器不 在属性序列的后面)，则返回true;否则返回false。

参见 next()。

QString QJSValueIterator::name() const

返回使用next()跳过的最后一个属性的名称。

参见 value()。

bool QJSValueIterator::next()

将迭代器向前推进一个位置。如果迭代器前面至少有一个项(即迭代器还没有位于属性序列的后面)，则返回true;否则返回false。

参见 hasNext() 和 name()。

QJSValue QJSValueIterator::value() const

返回使用next()跳过的最后一个属性的值。

See also name().

QJSValueIterator &QJSValueIterator::operator=(QJSValue &object)

使迭代器对object进行操作。迭代器被设置在属性序列的前面(在第一个属性之前)。
