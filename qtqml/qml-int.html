int QML基本类型

纯整数，例如： 0、 1、 或 -20。

int类型指的是一个整数，例如0、10或-20。

可能的int值范围从-2000000000到大约2000000000，尽管大多数类型只接受缩小的范围(他们在文档中提到了)。

示例：


  Item { width: 100; height: 200 }

这个基本类型由QML语言提供。

请参考 QML基本类型。
