QML的QAbstractState类型

QAbstractState类型是StateMachine的State的基类。更多…

Import Statement: import QtQml.StateMachine 1.12
Since: Qt 5.4
Inherited By:

FinalState, HistoryState, and State

  • 所有成员的列表，包括继承的成员

属性

  • active : bool

信号

  • entered()
  • exited()

详细描述

不要直接使用QAbstractState而是使用State, FinalState 或 StateMachine 。

参见 StateMachine 和 State.

属性文档

[read-only] active : bool

该状态的活动属性。entered() 和 exited() 信号之间的状态为活动状态。此属性是只读的。

参见 entered and exited.


信号文档

entered()

当状态激活时发出此信号。

相应的处理程序onEntered。

参见active（激活） 和 exited（退出）。


exited()

当状态变为非活动状态时发出此信号。

对应的处理程序是onExited.

参见 active 和 entered.
