QML的DelegateChoice类型

封装委托以及何时使用它。更多…

详细描述

DelegateChoice类型包装委托，并定义应该选择委托的环境。

DelegateChoices can be nested inside a DelegateChooser.

参见DelegateChooser.

属性文档

column : index

此属性包含用于匹配模型元素的column值的值。


delegate : Component

如果此选择与模型项匹配，则此属性保存委托以供使用


index : index

此属性包含用于匹配模型元素索引值的值。这实际上是row的别名。

参见row.


roleValue : string

此属性包含用于匹配委托DelegateChooser::role提供的角色的角色数据的值。


row : index

此属性包含用于匹配模型元素的行值的值。对于只有索引属性(因此只有一列)的模型，应该将此属性作为索引，并设置为所需的索引值。

注意:设置行和索引都有未定义的行为。这两个是等价的，应该只使用一个。

参见index。
