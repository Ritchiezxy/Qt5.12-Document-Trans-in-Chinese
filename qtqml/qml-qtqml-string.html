详细信息

QML String对象通过arg()函数扩展了JS String对象。

请参考 ECMAScript语言规范。

方法文档

string arg(value)

返回此字符串的副本，并将最低编号的位标记替换为值，即%1，%2，…, % 99。下面的示例打印出“There are 20 items”


  var message = "There are %1 items"
  var count = 20
  console.log(message.arg(count))
