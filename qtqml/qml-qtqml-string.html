详细描述

QML字符串对象使用arg() 函数扩展JS字符串对象。

请参见ECMAScript语言规范。

方法文档

string arg(value)

R返回该字符串的副本，用值替换编号最低的位置标记，即，., %1, %2, ..., %99。下面的示例打印"There are 20 items"


  var message = "There are %1 items"
  var count = 20
  console.log(message.arg(count))
