string QML Basic Type

一个自由格式的文本字符串。

string符串类型引用引号中的自由格式文本字符串，例如：. "Hello world!"。

例子


  Text { text: "Hello world!" }

字符串具有length 属性，该属性保存字符串中的字符数。

QML使用arg()函数扩展JavaScript字符串类型，以支持值替换。

当与C++集成时，请注意，从C++传递到QML的任何QString值都会自动转换为string 值，反之亦然。

这种基本类型由QML语言提供。

请参见QML基本类型。
