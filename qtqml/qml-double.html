double QML Basic Type

a number with a decimal point, stored in double precision.

The double type refers to a number with a decimal point and is stored in double precision, IEEE floating point format.

Example:


  Item {
      property double number: 32155.2355
  }

This basic type is provided by the QML language.

See also QML Basic Types.
