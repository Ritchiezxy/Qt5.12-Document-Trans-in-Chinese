所有PinchHandler成员的列表

这是PinchHandler的完整成员列表，包括继承的成员。

  • active : bool
  • centroid : QPointF
  • centroidVelocity : QVector2D
  • maximumRotation : real
  • maximumScale : real
  • maximumX : real
  • maximumY : real
  • minimumRotation : real
  • minimumScale : real
  • minimumTouchPoints : int
  • minimumX : real
  • minimumY : real
  • pinchOrigin : real
  • rotation : real
  • scale : real
  • translation : QVector2D

以下成员继承自MultiPointHandler。

  • acceptedDevices : int
  • acceptedModifiers : int
  • acceptedPointerTypes : int
  • active : bool
  • enabled : bool
  • grabPermission : bool
  • maximumPointCount : int
  • minimumPointCount : int
  • parent : Item
  • pointDistanceThreshold : real
  • target : Item
  • canceled(EventPoint point)
  • grabChanged(EventPoint point)
