Qt Quick Test QML Types

您可以使用以下语句导入此模块:


  import QtTest 1.11

有关如何使用这些类型的更多信息，请参见Qt Quick Test.

SignalSpy

Enables introspection of signal emission

TestCase

Represents a unit test case

TouchEventSequence

Used to build and dispatch touch events for testing
