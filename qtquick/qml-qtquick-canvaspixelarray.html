CanvasPixelArray QML 类型

提供对图像数据中每个像素的组件的有序和索引访问。 更多...

属性

  • length : int

详细信息

CanvasPixelArray对象提供了对图像数据中每个像素的颜色组件的有序索引访问。CanvasPixelArray可以作为普通的Javascript数组访问。

请参考 CanvasImageData 和 PixelArray的W3C 2d上下文标准。

属性文档

length : int

CanvasPixelArray对象表示h×w×4整数，w和h来自CanvasImageData。CanvasPixelArray对象的length属性必须返回h×w×4数字值。此属性为只读。
