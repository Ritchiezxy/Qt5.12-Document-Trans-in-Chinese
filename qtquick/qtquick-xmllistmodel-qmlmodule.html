Qt Quick XmlListModel QML Types

这个QML模块包含用于从XML数据创建模型的类型。

用此模块中的类型，请使用以下行导入模块:


  import QtQuick.XmlListModel 2.11

XmlListModel

For specifying a read-only model using XPath expressions

XmlRole

For specifying a role to an XmlListModel
