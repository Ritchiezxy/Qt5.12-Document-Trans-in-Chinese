所有PathAnimation成员的列表

这是PathAnimation的完整成员列表，包括继承的成员。

  • anchorPoint : point
  • duration : int
  • easing.amplitude : real
  • easing.bezierCurve : list<real>
  • easing.overshoot : real
  • easing.period : real
  • easing.type : enumeration
  • endRotation : real
  • orientation : enumeration
  • orientationEntryDuration : real
  • orientationExitDuration : real
  • path : Path
  • target : Item

以下成员继承自Animation。

  • alwaysRunToEnd : bool
  • loops : int
  • paused : bool
  • running : bool
  • complete()
  • pause()
  • restart()
  • resume()
  • start()
  • started()
  • stop()
  • stopped()
