所有TouchPoint成员的列表

这是TouchPoint的完整成员列表，包括继承的成员。

  • area : rectangle (obsolete)
  • ellipseDiameters : size
  • pointId : int
  • pressed : bool
  • pressure : real
  • previousX : real
  • previousY : real
  • rotation : real
  • sceneX : real
  • sceneY : real
  • startX : real
  • startY : real
  • uniqueId : PointingDeviceUniqueId
  • velocity : vector2d
  • x : real
  • y : real
