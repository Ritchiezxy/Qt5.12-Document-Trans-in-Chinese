QML的Drawer（抽屉）类型

侧板，可以通过滑动手势打开或关闭。更多…

Import Statement: import QtQuick.Controls 2.5
Since: Qt 5.7
Inherits:

Popup

  • 所有成员的列表，包括继承的成员。

属性

  • dragMargin : real

详细描述

Drawer提供了一个基于滑动的侧板，类似于那些经常用于触摸界面的侧板，该侧板为导航提供了一个集中位置。

Drawer项可以定位于内容项的任意四个边。上图中的抽屉靠窗的左边。然后把抽屉从窗口的左边缘"拖" 出来打开。


  import QtQuick 2.12
  import QtQuick.Controls 2.12

  ApplicationWindow {
      id: window
      visible: true

      Drawer {
          id: drawer
          width: 0.66 * window.width
          height: window.height

          Label {
              text: "Content goes here!"
              anchors.centerIn: parent
          }
      }
  }

Drawer是一种特殊类型的弹出窗口，位于窗口的一个edges。默认情况下，Drawer项将自己重设为窗口overlay（覆盖）的，从而对窗口坐标进行操作。 还可以手动将parent（父） 类设置为其他值，以使Drawer项在特定的坐标空间中运行。

Drawer可以配置为只覆盖其窗口边缘的一部分。下面的示例演示了如何将折叠项定位为显示在窗口标题下面:


  import QtQuick 2.12
  import QtQuick.Controls 2.12

  ApplicationWindow {
      id: window
      visible: true

      header: ToolBar { }

      Drawer {
          y: header.height
          width: window.width * 0.6
          height: window.height - header.height
      }
  }

position属性确定抽屉的可见程度，其值介于0.0 和 1.0之间。无法将抽屉的x坐标(或水平边距)设置在窗口的左右边缘，也无法将抽屉的y坐标(或垂直边距)设置在窗口的上下边缘。

在上图中，应用程序的内容被"推送"到屏幕上。这是通过对内容应用翻译实现的:


  import QtQuick 2.12
  import QtQuick.Controls 2.12

  ApplicationWindow {
      id: window
      width: 200
      height: 228
      visible: true

      Drawer {
          id: drawer
          width: 0.66 * window.width
          height: window.height
      }

      Label {
          id: content

          text: "Aa"
          font.pixelSize: 96
          anchors.fill: parent
          verticalAlignment: Label.AlignVCenter
          horizontalAlignment: Label.AlignHCenter

          transform: Translate {
              x: drawer.position * content.width * 0.33
          }
      }
  }

如果您希望应用程序的内容在抽屉打开时保持原样，则不要使用翻译功能。

通过使Drawer项非模态 和非交互式，可以将Drawer项配置为非封闭的持久侧面板。有关详细信息，请参阅 侧面板示例。

注意: 在某些平台上，某些边可能被保留为系统手势，因此不能与Drawer一起使用。例如，顶部和底部边缘可以预留给Android和iOS上的系统通知和控制中心。

属性的文档

dragMargin : real

此属性保存从屏幕边缘到拖动操作将打开Drawer项的屏幕边缘的距离。将值设置为0或更小，可以防止通过拖动打开抽屉。

默认值是 Qt.styleHints.startDragDistance。

参见interactive（交互）。


edge : enumeration

此属性保存抽屉被放在哪个边缘。可接受的数值为:

ConstantDescription
Qt.TopEdge窗户的上边缘。
Qt.LeftEdge窗口的左边缘(默认)。
Qt.RightEdge窗户的右边。
Qt.BottomEdge窗户的底边。

interactive : bool

此属性保存抽屉是否具有交互性。非交互式抽屉不会对滑动做出反应。

默认值为true。

这个属性是在 QtQuick.Controls 2.2 (Qt 5.9)中引入的。

参见dragMargin。/p>


position : real

此属性保存抽屉相对于其最终目的地的位置。当抽屉完全关闭时的位置为0.00.0，完全打开时的位置为1.0。
